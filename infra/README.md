# Infra

Here is the list of **docker definition** files for each services that run in the production environment. 

- `app.yaml`: pokesocket app core services definition.
- `nginx.yaml`: Proxy definition for handling internet traffic.
- `portainer.sh`: Remote Docker environment manager.
- `wud.yml`: Docker image autoupdater.
